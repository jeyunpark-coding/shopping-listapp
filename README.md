# 🛒 쇼핑 리스트 앱

아이템 추가, 삭제, 체크 기능이 있는 간단한 쇼핑 리스트 웹앱입니다.
외부 라이브러리 없이 순수 HTML / CSS / JavaScript로 만들어졌습니다.

## 기능

- **아이템 추가** — 입력창에 텍스트 입력 후 `추가` 버튼 클릭 또는 `Enter` 키
- **체크/해제** — 체크박스 클릭으로 완료/미완료 토글 (취소선 표시)
- **단일 삭제** — 각 항목 우측 `✕` 버튼
- **완료 항목 일괄 삭제** — 하단 "완료 항목 삭제" 버튼
- **필터** — 전체 / 미완료 / 완료 탭으로 필터링
- **자동 저장** — `localStorage`에 저장되어 새로고침 후에도 유지

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

```
브라우저에서 index.html 더블클릭
```

## 기술 스택

- HTML5
- CSS3 (Flexbox, CSS Animation, accent-color)
- Vanilla JavaScript (ES6+, localStorage)
